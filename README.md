Ellipsis is a straightforward self-hosted identity solution based on
OpenID Connect. It supports the following features:

* [x] Single Sign-On
* [x] Session management
* [x] Back-channel logout
* [x] Social logins
* [x] Authorization code flow
* [ ] Authorization code flow with PKCE (coming soon)
